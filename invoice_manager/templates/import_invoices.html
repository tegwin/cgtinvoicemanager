Import Invoices from CSV

Each row will create one invoice with one line item.

Required columns:

  • issue_date (YYYY-MM-DD)
  • due_date (YYYY-MM-DD)
  • item_description
  • item_quantity
  • item_unit_price

Customer columns:

  • customer_email (recommended; used to find or create the customer)
  • customer_name (required if email is missing)

Optional columns:

  • status (draft|sent|paid|overdue|cancelled, default: sent)
  • notes
  • invoice_number (if blank, a number is auto-generated)
